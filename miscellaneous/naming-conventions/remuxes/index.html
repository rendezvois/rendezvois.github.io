    Naming Conventions for Remuxes#

    This page contains only examples. For more details about the elements in the file names, look at the Overview page.

    High Definition (HD)#

    Type Filename
    DTS-HD MA audio 
    Movie.Title.2010.1080p.Blu-ray.Remux.AVC.DTS-HD.MA.5.1-TAG
    FLAC audio 
    Movie.Title.2010.1080p.Blu-ray.Remux.AVC.FLAC2.0-TAG
    TrueHD audio 
    Movie.Title.2010.1080p.Blu-ray.Remux.AVC.TrueHD.5.1-TAG
    TrueHD Atmos audio 
    Movie.Title.2010.1080p.Blu-ray.Remux.AVC.TrueHD.7.1.Atmos-TAG
    MPEG-2 video 
    Movie.Title.2010.1080p.Blu-ray.Remux.MPEG-2.DTS-HD.MA.5.1-TAG
    VC-1 video 
    Movie.Title.2010.1080p.Blu-ray.Remux.VC-1.DTS-HD.MA.5.1-TAG
    1080i video 
    Movie.Title.2010.1080i.Blu-ray.Remux.AVC.DTS-HD.MA.5.1-TAG
    Distributor 
    Movie.Title.2010.Criterion.Collection.1080p.Blu-ray.Remux.AVC.FLAC1.0-TAG
    Region 
    Movie.Title.2010.1080p.GER.Blu-ray.Remux.AVC.TrueHD.7.1-TAG
    Different Cut 
    Movie.Title.2010.Directors.Cut.1080p.Blu-ray.Remux.AVC.DTS-HD.MA.5.1-TAG
