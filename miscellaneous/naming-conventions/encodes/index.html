    Naming Conventions for Encodes#

    This page contains only examples. For more details about the elements in the file names, look at the Overview page.

    DVDRips#

    Type Filename
    Default 
    Movie.Title.2010.DVDRip.x264-TAG
    NTSC 
    Movie.Title.2010.NTSC.DVDRip.x264-TAG
    PAL 
    Movie.Title.2010.PAL.DVDRip.x264-TAG
    Foreign Movie 
    Original.Title.2010.AKA.English.Title.DVDRip.x264-TAG
    Different Cut 
    Movie.Title.2010.Directors.Cut.DVDRip.x264-TAG
    Repack 
    Movie.Title.2010.REPACK.DVDRip.x264-TAG
    With Audio 
    Movie.Title.2010.DVDRip.DD5.1.x264-TAG
    		 
    Movie.Title.2010.DVDRip.DD2.0.x264-TAG
